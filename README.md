This respository includes the software portion of ATSU, the Affordable Thermal Sensing Unit. 
If you would like to build your own, you need to aquire an M5StackC Plus 2 and a M5Stack Thermal 2 module.
Connect them together with the cable included with the Thermal2 module and use Arduino IDE or PlatformIO to flash the firmware found in this repo to the M5StickC Plus. 

How to use:

Cycle colour schema: single-press middle button (Button A)
Change brightness: single-press Button B
Night Mode: double-press Button B
Toggle flashlight: single-press Button B while in Night Mode
Disable Wi-Fi: press and hold Button B when Wi-Fi is connected to an AP
Hotspot Mode: press and hold Button B when not connected to an AP
Dowsing Mode: single-press Button C
